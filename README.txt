# نظام إدارة مدونة بسيطة

## المتطلبات
- PHP 7.4 أو أحدث
- MySQL 5.7 أو أحدث
- خادم ويب (مثل Apache أو Nginx)

## إعداد قاعدة البيانات
1. قم بإنشاء قاعدة بيانات جديدة باسم `blog_db`:
    ```sql
    CREATE DATABASE blog_db;
    ```

2. بعد اعداد اتصال مع قاعدة البيانات قم بتشغيل الملف localhost.sql ليتم انشاء الجداول posts,users

## إعداد المشروع
1. قم بنسخ ملفات المشروع إلى مجلد خادم الويب الخاص بك ( `htdocs` في XAMPP ).
  

## تشغيل المشروع
1. افتح متصفح الويب وانتقل إلى الرابط التالي :`http://localhost/SQL_OOP_TASK/`
سيتم الانتقال لصفحة index.php  وهي عبارة عن login form
في حال وجود حساب قم بتسجيل الدخول باستخدام user name and password
 في حال عدم وجود حساب قم بتسجيل حساب جديد باستخدام register


    - لإنشاء مقالة جديدة: `add_post.php`
    - لعرض جميع المقالات: `show_posts.php`
    - لعرض مقالة واحدة: `show_one_post.php` 
    - لتعديل مقالة: `edit_post.php` 
    - لحذف مقالة: `delete_post.php`
